# heroku-rails-5

# Deployment instructions
## Setup your local
```
$ git clone git@github.com:tayutaedomo/heroku-rails-5.git
$ cd heroku-rails-5
$ bundle install --path vendor/bundle
$ bin/rake db:create
$ bin/rake db:migrate
$ bin/rails s
```

Your app should now be running on [http://localhost:3000](http://localhost:3000).
